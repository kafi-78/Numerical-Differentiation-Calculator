Numerical Differentiation Theory

What is Numerical Differentiation?

Numerical Differentiation is a method used to approximate the derivative of a function using nearby data points. It is especially useful when the exact derivative is difficult to calculate or the function is only available as numerical data.


Forward Difference

Formula:

f'(x) = ( f(x+h) - f(x) ) / h

Uses the function value at x and a nearby point x+h.

Theory:
Forward Difference estimates the derivative using the current point and a nearby forward point. It is simple to implement but generally less accurate than the Central Difference method.


Backward Difference

Formula:

f'(x)= ( f(x) - f(x-h) ) / h

Uses values behind the point.

Theory:
Backward Difference estimates the derivative using the current point and a nearby backward point. It is useful when future data points are unavailable.


Central Difference

Formula:

f'(x)= ( f(x+h) - f(x-h) ) / (2h)

Provides higher accuracy by using values on both sides.

Theory:
Central Difference uses points on both sides of the target value. Because it considers information from both directions, it usually provides the highest accuracy among basic methods.


Divided Difference

Formula:

f'(x)= ( f(x+h) - f(x) ) / h

Widely used in interpolation and approximation methods.

Theory:
Divided Difference is based on interpolation concepts and measures the rate of change between two nearby points. It is commonly used in numerical analysis and approximation.
