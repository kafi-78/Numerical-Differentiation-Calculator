Tutorial & User Guide

Step 1

Enter the mathematical function.

Example: x^3 + 2*x


Step 2

Enter the value of x where the derivative will be calculated.

Example: x = 2


Step 3

Enter the step size h.

Example: h = 0.01


Step 4

Choose a differentiation method:

  • Forward Difference
  • Backward Difference
  • Central Difference
  • Divided Difference

Step 5

Click the Calculate button.

The application will show:

  • Estimated Derivative
  • Exact Derivative
  • Error
  • Function Graph
  • Error Graph
  • Comparison Table

Input Rules

  • Function must use the variable x.
  • Use * for multiplication.
  • Use ^ for powers.
  • Step size h should be a small positive value.
  • Do not leave input fields empty.

Supported Functions

  • x^2
  • x^3 + 2*x
  • sin(x)
  • cos(x)
  • exp(x)
  • log(x)

Features

  • Forward Difference Method
  • Backward Difference Method
  • Central Difference Method
  • Divided Difference Method
  • Function Visualization
  • Error Analysis Graph
  • Method Comparison Table
  • PDF Export
  • PNG Graph Download

Example Problem

Function: x^3 + 2*x

x = 2

h = 0.01
