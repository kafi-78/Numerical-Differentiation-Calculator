About Project

Project Title

Numerical Differentiation Calculator


Project Objective

This project is designed to estimate the derivative of mathematical functions using various numerical differentiation methods.

It helps students understand how different numerical methods work and how the step size affects accuracy.


Features

  • Forward Difference Method
  • Backward Difference Method
  • Central Difference Method
  • Divided Difference Method
  • Function Visualization
  • Error Analysis
  • Method Comparison
  • Educational Resources
  • Tutorial Section

Technologies Used

  • HTML5
  • CSS3
  • Bootstrap 5
  • JavaScript
  • Math.js
  • Plotly.js
